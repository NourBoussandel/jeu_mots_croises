La programmation web et multimédia couvre les concepts de base et
la syntaxe des langages de programmation permettant la mise en œuvre des sites web statiques et dynamiques.
Parmi les langages , on distingue :
-HTML : Hypertext Markup Language connu par « langage de balises » donnant des informations sur la structure
et la mise en forme de texte .
Ses principales balises sont html , head et body.
-CSS : Cascading Style Sheet : complète le HTML afin de mettre en forme vos sites web et leur donner l’apparence souhaitée.
Le code CSS est soit inséré :
-Dans un fichier indépendant appelé en général style.css
-inséré directement dans une balise style à l’intérieur de l’entête head
comme suit "link rel= «stylesheet » href= « style.css »"
-JS : Java Script : permet d’implémenter des fonctions pour manipuler les
évènements de la souris , les messages d’alerte et les données des formulaires.
-PHP : Personnal Home Page permet de créer des pages web dynamiques et interactives via le serveur http.
